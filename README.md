# TaxMate NG

A simple, clear tax companion for Nigerians. Built with Next.js, TypeScript, and Tailwind CSS. Includes a calculator, visual dashboards, and a learning hub that explains the 2026 Nigerian tax changes in plain English.

## Overview

- Understand tax rules quickly with human-friendly summaries
- Estimate taxes with a guided calculator
- Explore the Education Hub for the 2026 reforms
- Clean, responsive UI with a focus on clarity

## Tech Stack

- Next.js 16 (App Router)
- React 19 and TypeScript
- Tailwind CSS v4
- Radix UI for accessible components
- Framer Motion for subtle animations

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm

### Install

```bash
pnpm install
# or
npm install
```

### Develop

```bash
pnpm dev
# or
npm run dev
```

App runs at http://localhost:3000

### Lint

```bash
pnpm lint
# or
npm run lint
```

## Project Structure

- app: Next.js routes and layout
- components: UI and feature components
- lib: shared utilities
- public: static assets

Key routes:

- /: Home, calculator, dashboard, reforms overview
- /education-hub: Detailed, plain-English guide to 2026 tax changes

## Features

- Tax Calculator: Estimates based on current rules with clear breakdowns
- Visual Dashboard: Highlights key numbers and progress
- Reform Knowledge Base: Concise summaries of major changes
- Education Hub: Searchable cards and audience-focused guidance
